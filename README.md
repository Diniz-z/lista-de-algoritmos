# Lista de Exercícios - Algoritmos

Este repositório contém a resolução dos exercícios de treinamento para a avaliação da faculdade. Os códigos foram desenvolvidos em **C** utilizando a IDE CLion.

## Estrutura do Repositório

* **Lista 01**: Exercícios básicos de lógica, entrada e saída de dados.
* **lista 02**: Estruturas condicionais e de repetição.
* **lista 03 (recursividade)** & **lista 04 (vh)**: Manipulação de vetores e funções.
